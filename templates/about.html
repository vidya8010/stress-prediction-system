About This Project

This project is developed to predict stress levels of users based on their social media usage patterns. It aims to create awareness about digital well-being and mental health.

My Mission

To empower individuals by providing insights into how online behavior influences stress, and to promote balanced usage of social platforms.

Technologies Used

  • Python (scikit-learn, Pandas, NumPy)
  • Flask Web Framework
  • Bootstrap for Frontend Styling
